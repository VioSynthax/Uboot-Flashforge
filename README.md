# Uboot Flashforge
#### Uboot for Flashforge Adventurer 3 / Monoprice Voxel (with Widora web recovery)
***
# How to use
* 1. make menuconfig
* 2. select MT7628 board
* 3. make clean;make

### Note
* compilation requires java such as 1.7.0_79

* based on https://github.com/widora/u-boot-mt7688
